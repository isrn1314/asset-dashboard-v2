# AGENTS.md

## 專案定位
- 這是一個以 HTML / CSS / JS 製作的個人資產總覽 / 資產管理小系統
- 目標是超實用、可長期使用、可持續擴充
- 優先重視正確性、穩定性、可維護性、手機可用性
- 不追求花俏展示效果

## 開發原則
- 複雜任務先規劃，再動手改檔
- 非必要不要新增依賴
- 優先維持簡單、清楚、可維護的結構
- 每次修改都要考慮 localStorage 相容性、資料遺失風險、手機版顯示、匯入匯出相容性
- 不可為了快而犧牲結構品質
- 修改後要自查

## 修改規範
- 除非我明確要求，否則不要一次大改很多檔案
- 先說明計畫，再執行
- 若要新增檔案，先說明用途
- 若要改資料結構，必須先說明舊資料怎麼相容
- 若有風險，必須先提示

## 目前專案重點
- index.html
- style.css
- script.js
- manifest.webmanifest
- app-icon.svg

## 回覆格式要求
每次協助此專案時，請盡量用以下格式：
- 目標
- 變更計畫
- 風險 / 注意事項
- 再開始執行
