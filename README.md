# Codex資產網頁

## 專案簡介
這是一個用 HTML / CSS / JS 製作的個人資產總覽 / 資產管理小系統。

目前目標：
- 超實用
- 可長期使用
- 可持續擴充
- 手機可用
- 結構清楚、可維護

## 目前已有功能
- 資產總覽頁面
- localStorage 儲存
- 匯入 / 匯出 / 重置
- 響應式設計
- manifest / PWA 基礎檔案
